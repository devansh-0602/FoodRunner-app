# FoodRunner-App

It is an online food ordering app which allows the users to order food 
of their choice from the available restaurant in their area.

## The features are: 
### Splash screen
### Login screen 
     Login credentials
     Register yourself
     Forgot password
### Menu drawer
     Menu header
     Registered Name
     Registered Phone number
     
     1) Home: List of restaurants
             Sort By:
               Cost (Low to High)
               Cost (High to Low)
               Rating
     2) My profile
     3) Favourite Restaurants
     4) Order History
     5) FAQs
     6) Log out
